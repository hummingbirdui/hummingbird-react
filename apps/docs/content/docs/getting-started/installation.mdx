Welcome to the **Hummingbird React** documentation!

This site is powered by `fumadocs-mdx` for the data layer, and uses a fully custom layout built with Hummingbird components.

## Interactive Components

You can embed Hummingbird React components directly in your MDX files. For example, here is a custom button:

<Button variant="filled" color="primary">
  Click Me
</Button>

## Features

- **Fast & Lightweight:** Optimized Next.js App Router performance.
- **Custom UI:** Complete control over your layouts.
- **Type-safe:** Frontmatter and routing are fully typed.

Feel free to create more folders and `.mdx` files inside `content/docs` to expand this documentation!
