## With an image

`CardImage` places an image whose `position` controls which corners are rounded.

## Image overlay

`CardImageOverlay` lays the `CardBody` over the image.

<ComponentPreview name="card-image-overlay" />

## Horizontal

The `aside` prop lays the card out as a row.

<ComponentPreview name="card-aside" />

## Clickable

The `action` prop makes the whole card an interactive hover surface, paired with `asChild` to render as a link.

<ComponentPreview name="card-action" />

## Card group

`CardGroup` joins cards into an attached row.

<ComponentPreview name="card-group" />

## Styling

Hummingbird React card is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available card classes in the [Class overview](https://hbui.dev/docs/components/card/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/card/#css-variables) documentation to explore all available variables.
