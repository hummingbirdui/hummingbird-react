## Default

<ComponentPreview name="nav-default" />

## Tabs

The `tabs` variant renders the links as a bordered tab bar.

<ComponentPreview name="nav-tabs" />

## Underline

The `underline` variant underlines the active link.

<ComponentPreview name="nav-underline" />

## Colors

The `color` prop themes the whole nav surface with `primary`, `secondary`, `info`, `success`, `warning`, `danger`, or `neutral`.

<ComponentPreview name="nav-colors" />

## Active

The `active` prop on `Nav.Link` marks the current link and sets <code>aria-current="page"</code>.

<ComponentPreview name="nav-active" />

## Disabled

The `disabled` prop on `Nav.Link` mutes the link and stops interaction.

<ComponentPreview name="nav-disabled" />

### Nav.Item

`Nav.Item` is a layout wrapper that accepts only `className`.

## Styling

Hummingbird React nav is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available nav classes in the [Class overview](https://hbui.dev/docs/components/nav/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/nav/#css-variables) documentation to explore all available variables.
