---
title: Pagination
description: Navigation that splits content across a sequence of numbered pages.
---

## Default

`Pagination` is an accessible `<nav>` wrapping a `Pagination.Content` list of `Pagination.Item`s, with the current page marked by `active`.

<ComponentPreview name="pagination-default" className="flex justify-center" />

## Sizes

The `size` prop on `Pagination` offers `sm`, `md` (default), and `lg`.

<ComponentPreview name="pagination-sizes" className="flex justify-center" />

## Circle

The `shape="circle"` prop on `Pagination` renders rounded page links.

<ComponentPreview name="pagination-circle" className="flex justify-center" />

## Variants

The `variant` prop on `Pagination` sets the active-page treatment: `filled` (default), `subtle`, or `outlined`.

<ComponentPreview name="pagination-variants" className="flex justify-center" />

## Colors

The `color` prop on `Pagination` conveys intent with `primary` (default), `secondary`, `info`, `success`, `warning`, `danger`, or `neutral`.

<ComponentPreview name="pagination-colors" className="flex justify-center" />

## Disabled

The `disabled` prop on `Pagination.Item` mutes and deactivates a control such as the previous link.

<ComponentPreview name="pagination-disabled" className="flex justify-center" />

## Styling

Hummingbird React pagination is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available pagination classes in the [Class overview](https://hbui.dev/docs/components/pagination/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/pagination/#css-variables) documentation to explore all available variables.
