---
title: Pagination
description: Navigation that splits content across a sequence of numbered pages.
---

## Default

`Pagination` is an accessible `<nav>` wrapping a `PaginationContent` list of `PaginationItem`s, with the current page marked by `active`.

<ComponentPreview name="pagination-default" />

## Sizes

The `size` prop on `PaginationContent` offers `sm`, `md` (default), and `lg`.

<ComponentPreview name="pagination-sizes" />

## Circle

The `shape="circle"` prop on `PaginationContent` renders rounded page links.

<ComponentPreview name="pagination-circle" />

## Variants

The `variant` prop on `PaginationContent` sets the active-page treatment: `filled` (default), `subtle`, or `outlined`.

<ComponentPreview name="pagination-variants" />

## Colors

The `color` prop on `PaginationContent` conveys intent with `primary` (default), `secondary`, `info`, `success`, `warning`, `danger`, or `neutral`.

<ComponentPreview name="pagination-colors" />

## Disabled

The `disabled` prop on `PaginationItem` mutes and deactivates a control such as the previous link.

<ComponentPreview name="pagination-disabled" />

## With icons

`PaginationLink` accepts icon children for previous and next controls.

<ComponentPreview name="pagination-with-icons" />

## Styling

Hummingbird React pagination is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available pagination classes in the [Class overview](https://hbui.dev/docs/components/pagination/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/pagination/#css-variables) documentation to explore all available variables.
