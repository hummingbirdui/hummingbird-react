## Default

<ComponentPreview name="drawer-default" className="text-center" />

## Direction

Passing `direction` (`"top" | "right" | "bottom" | "left"`) to `Drawer` sets the edge the panel slides from.

<ComponentPreview name="drawer-direction" />

## Without overlay

Setting `overlay={false}` on `DrawerContent` renders the panel without the backdrop.

<ComponentPreview name="drawer-without-overlay" className="text-center" />

## Scrollable body

`DrawerBody` grows to fill the panel and scrolls its own overflow while the header stays pinned.

<ComponentPreview name="drawer-scrollable" className="text-center" />

## Styling

Hummingbird React drawer is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available drawer classes in the [Class overview](https://hbui.dev/docs/components/drawer/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/drawer/#css-variables) documentation to explore all available variables.
