title: Avatar
description: An image element with a text fallback for representing a user or entity.
---

## Default

An avatar renders an image with a fallback for when the image is unavailable.

<ComponentPreview name="avatar-default" />

## Sizes

The `size` prop offers four sizes: `xs`, `sm`, `md` (default), and `lg`.

<ComponentPreview name="avatar-sizes" />

## Status

The `status` prop adds a presence indicator: `online`, `offline`, `away`, or `busy`.

<ComponentPreview name="avatar-status" />

## Fallback

`AvatarFallback` renders when the image is unavailable — typically the user's initials.

<ComponentPreview name="avatar-fallback" />

## Group

`AvatarGroup` stacks several avatars into an overlapping row.

<ComponentPreview name="avatar-group" />

## API Reference

`Avatar`, `AvatarImage`, and `AvatarFallback` wrap the Radix UI Avatar primitives — see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/avatar#api-reference) documentation for their props.

## Styling

Hummingbird React avatar is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available avatar classes in the [Class overview](https://hbui.dev/docs/components/avatar/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/avatar/#css-variables) documentation to explore all available variables.
