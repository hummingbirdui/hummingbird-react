---
title: Radio
description: A styled native radio button, with an optional group wrapper.
---

### Default

Wrap radios in a `RadioGroup` to share a `name` and manage the selected value.

<div className="my-4">
  <RadioGroup name="plan" defaultValue="pro" className="flex flex-col gap-2">
    <Radio value="free" label="Free" />
    <Radio value="pro" label="Pro" />
    <Radio value="team" label="Team" />
  </RadioGroup>
</div>
