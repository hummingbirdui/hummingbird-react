title: Switch
description: A toggle switch built on a native checkbox with role="switch".
---

### Default

Pass a `label` to render a complete, clickable field.

<div className="flex flex-col gap-2 my-4">
  <Switch label="Enable notifications" defaultChecked />
</div>
