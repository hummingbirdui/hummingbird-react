## API Reference

For detailed usage guidelines, see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/popover#api-reference) documentation.

`PopoverHeader` and `PopoverBody` render the popover's header and body regions. Each accepts a `className`.

## Styling

Hummingbird React popover is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available popover classes in the [Class overview](https://hbui.dev/docs/components/popover/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/popover/#css-variables) documentation to explore all available variables.
