## Default

The default alert is filled with the primary color.

<ComponentPreview name="alert-default" />

## Variants

The `variant` prop changes the visual style: `filled` (default), `subtle`, or `outline`.

<ComponentPreview name="alert-variants" />

## Colors

The `color` prop conveys intent with `primary` (default), `neutral`, `secondary`, `info`, `success`, `warning`, or `danger`.

<ComponentPreview name="alert-colors" />

## With icon

`Alert.Icon` renders a leading icon.

## Styling

Hummingbird React alert is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available alert classes in the [Class overview](https://hbui.dev/docs/components/alert/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/alert/#css-variables) documentation to explore all available variables.
