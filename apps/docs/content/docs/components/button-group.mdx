---
title: Button Group
description: Group a series of buttons together on a single line or stack.
---

### Default

Wrap a set of `Button`s in a `ButtonGroup` to join them into a single horizontal control.

<div className="flex flex-wrap items-center gap-2 my-4">
  <ButtonGroup>
    <Button variant="outline">Left</Button>
    <Button variant="outline">Middle</Button>
    <Button variant="outline">Right</Button>
  </ButtonGroup>
</div>

### Sizes

Set `size` to `sm`, `md` (default), or `lg` to scale every button in the group.

<div className="flex flex-col items-start gap-3 my-4">
  <ButtonGroup size="sm">
    <Button variant="outline">Left</Button>
    <Button variant="outline">Middle</Button>
    <Button variant="outline">Right</Button>
  </ButtonGroup>
  <ButtonGroup size="md">
    <Button variant="outline">Left</Button>
    <Button variant="outline">Middle</Button>
    <Button variant="outline">Right</Button>
  </ButtonGroup>
  <ButtonGroup size="lg">
    <Button variant="outline">Left</Button>
    <Button variant="outline">Middle</Button>
    <Button variant="outline">Right</Button>
  </ButtonGroup>
</div>

### Vertical

Set `orientation="vertical"` to stack the buttons instead.

<div className="flex flex-wrap items-start gap-2 my-4">
  <ButtonGroup orientation="vertical">
    <Button variant="outline">Top</Button>
    <Button variant="outline">Middle</Button>
    <Button variant="outline">Bottom</Button>
  </ButtonGroup>
</div>

### Colors

Each button keeps its own `variant` and `color`, so groups can mix intents.

<div className="flex flex-wrap items-center gap-2 my-4">
  <ButtonGroup>
    <Button color="primary">Primary</Button>
    <Button color="success">Success</Button>
    <Button color="danger">Danger</Button>
  </ButtonGroup>
</div>

### Toolbar

Use `ButtonToolbar` to combine multiple groups into a single toolbar.

<div className="flex flex-wrap items-center gap-2 my-4">
  <ButtonToolbar className="gap-2">
    <ButtonGroup>
      <Button variant="outline">1</Button>
      <Button variant="outline">2</Button>
      <Button variant="outline">3</Button>
    </ButtonGroup>
    <ButtonGroup>
      <Button variant="outline">4</Button>
      <Button variant="outline">5</Button>
    </ButtonGroup>
    <ButtonGroup>
      <Button variant="outline">6</Button>
    </ButtonGroup>
  </ButtonToolbar>
</div>
