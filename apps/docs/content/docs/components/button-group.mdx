---
title: Button Group
description: Groups a series of buttons together on a single line or stack.
---

## Default

Wrapping a set of `Button`s in a `ButtonGroup` joins them into a single horizontal control.

<ComponentPreview name="button-group-default" />

## Vertical

Setting `orientation="vertical"` stacks the buttons instead.

<ComponentPreview name="button-group-vertical" />

## Sizes

The `size` prop scales every button in the group: `sm`, `md` (default), or `lg`.

<ComponentPreview name="button-group-sizes" />

## Toolbar

`ButtonToolbar` combines multiple groups into a single toolbar.

<ComponentPreview name="button-group-toolbar" />

## Styling

Hummingbird React button group is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available button group classes in the [Class overview](https://hbui.dev/docs/components/button-group/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/button-group/#css-variables) documentation to explore all available variables.
