title: Tabs

## Default

`TabsList` uses the bordered style by default. Set `defaultValue` to select the initial tab.

<ComponentPreview name="tabs-default" />

## Underline

Pass `variant="underline"` to `TabsList` for the underlined tab style.

<ComponentPreview name="tabs-underline" />

## Plain

Pass `variant="default"` to `TabsList` for a borderless list of triggers.

<ComponentPreview name="tabs-plain" />

## Colors

The `color` prop on `TabsList` tints the active trigger.

<ComponentPreview name="tabs-colors" />

## Disabled

Add `disabled` to a `TabsTrigger` to prevent its panel from being selected.

<ComponentPreview name="tabs-disabled" />

## Styling

Hummingbird React tabs is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available tabs classes in the [Class overview](https://hbui.dev/docs/components/tabs/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/tabs/#css-variables) documentation to explore all available variables.
