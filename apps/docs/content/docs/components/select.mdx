---
title: Select
description: A styled native select for choosing one option from a list.
---

### Default

`Select` renders a native `<select>`. Pass `<option>`s as children.

<div className="flex flex-col gap-2 max-w-sm my-4">
  <Select defaultValue="">
    <option value="" disabled>
      Choose a fruit…
    </option>
    <option value="apple">Apple</option>
    <option value="banana">Banana</option>
    <option value="cherry">Cherry</option>
  </Select>
</div>

### Variants

<div className="flex flex-col gap-2 max-w-sm my-4">
  <Select variant="outline" defaultValue="apple">
    <option value="apple">Outline</option>
  </Select>
  <Select variant="fill" defaultValue="apple">
    <option value="apple">Fill</option>
  </Select>
</div>
