---
title: Form Range
description: A styled native range slider for selecting a value from a range.
---

### Default

`FormRange` renders a native `<input type="range">`.

<div className="max-w-sm my-4">
  <FormRange defaultValue={50} />
</div>

### Sizes

<div className="flex flex-col gap-4 max-w-sm my-4">

  <FormRange size="md" defaultValue={50} />
  <FormRange size="lg" defaultValue={60} />
</div>
