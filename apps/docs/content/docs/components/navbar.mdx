## Default

The default navbar composes a `NavbarBrand` and a `NavbarNav` of `NavLink`s.

<ComponentPreview name="navbar-default" />

## Expand

The `expand` prop sets the breakpoint at which the bar lays out horizontally: `sm`, `md`, `lg`, `xl`, `2xl`, or `always`.

<ComponentPreview name="navbar-expand" />

## Responsive collapse

`NavbarToggle` and `NavbarCollapse` hide the menu behind a toggle below the `expand` breakpoint.

<ComponentPreview name="navbar-collapse" />

## Always collapsed

Omitting `expand` keeps the menu behind the toggle at every width, and `duration` on `NavbarCollapse` tunes the open and close speed in milliseconds.

<ComponentPreview name="navbar-always-collapsed" />

## Scrollable

The `scrollable` prop on `NavbarNav` constrains the list height and scrolls its overflow.

<ComponentPreview name="navbar-scrollable" />

## With text

`NavbarText` renders non-link content such as a signed-in label.

<ComponentPreview name="navbar-text" />

## Custom toggle

Passing children to `NavbarToggle` overrides the default hamburger glyph.

<ComponentPreview name="navbar-custom-toggle" />

### NavbarToggle

`NavbarToggle` wraps the Radix [Collapsible](https://www.radix-ui.com/primitives/docs/components/collapsible#api-reference) trigger. For detailed usage guidelines, see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/collapsible#api-reference) documentation.

### NavbarBrand and NavbarText

`NavbarBrand` and `NavbarText` are layout wrappers that accept only `asChild` and `className`. `NavbarTogglerIcon` renders the default hamburger glyph and accepts standard `svg` props.

## Styling

Hummingbird React navbar is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available navbar classes in the [Class overview](https://hbui.dev/docs/components/navbar/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/navbar/#css-variables) documentation to explore all available variables.
