---
title: Close Button
description: An icon-only button for dismissing content.
---

### Default

`CloseButton` renders an × icon button. It defaults to `type="button"` and
`aria-label="Close"`.

<div className="flex items-center gap-3 my-4">
  <CloseButton />
</div>

### Circle

<div className="flex items-center gap-3 my-4">
  <CloseButton shape="circle" />
</div>

### On a dark surface

Use `variant="white"` so the icon stays visible on dark backgrounds.

<div className="flex items-center gap-3 my-4 p-4 rounded-md bg-neutral-900">
  <CloseButton variant="white" />
  <CloseButton variant="white" shape="circle" />
</div>

### Disabled

<div className="flex items-center gap-3 my-4">
  <CloseButton disabled />
</div>
