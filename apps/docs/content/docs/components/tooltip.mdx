## Default

<ComponentPreview name="tooltip-default" className="text-center" />

## Placement

Pass `side` (`"top" | "right" | "bottom" | "left"`) to `Tooltip.Content` to choose which edge of the trigger the tooltip points from.

<ComponentPreview name="tooltip-placement" />

## Delay

Each `Tooltip` wraps its own provider, so `delayDuration` on the `Tooltip` tunes how long the pointer must rest before it opens.

<ComponentPreview name="tooltip-delay" />

## Provider

Mounting a single `Tooltip.Provider` shares one `delayDuration` and the skip-delay behavior across every tooltip beneath it.

<ComponentPreview name="tooltip-provider" />

## API Reference

Built on the [Radix UI Tooltip](https://www.radix-ui.com/primitives/docs/components/tooltip) primitive. Each part forwards all props to its Radix counterpart.

### Tooltip.Provider

Wraps your app to provide global functionality to your tooltips. Renders no DOM element of its own.

### Tooltip

Contains all the parts of a tooltip. Each `Tooltip` wraps itself in its own `Tooltip.Provider` with a `delayDuration` of `0`, so tooltips open instantly unless a delay is set. Renders no DOM element of its own.

## Styling

Hummingbird React tooltip is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available tooltip classes in the [Class overview](https://hbui.dev/docs/components/tooltip/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/tooltip/#css-variables) documentation to explore all available variables.
