## Placement

Pass `side` (`"top" | "right" | "bottom" | "left"`) to `TooltipContent` to choose which edge of the trigger the tooltip points from.

<ComponentPreview name="tooltip-placement" />

## Delay

Each `Tooltip` wraps its own provider, so `delayDuration` on the `Tooltip` tunes how long the pointer must rest before it opens.

<ComponentPreview name="tooltip-delay" />

## Provider

Mounting a single `TooltipProvider` shares one `delayDuration` and the skip-delay behavior across every tooltip beneath it.

<ComponentPreview name="tooltip-provider" />

## Styling

Hummingbird React tooltip is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available tooltip classes in the [Class overview](https://hbui.dev/docs/components/tooltip/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/tooltip/#css-variables) documentation to explore all available variables.
