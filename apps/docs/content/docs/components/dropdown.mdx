## API Reference

Built on the [Radix UI Dropdown Menu](https://www.radix-ui.com/primitives/docs/components/dropdown-menu) primitive. Each part forwards all props to its Radix counterpart.

### Dropdown.ItemText

Renders non-interactive text inside the menu as a `<span>`. This is a Hummingbird addition with no Radix counterpart; it accepts all standard `<span>` props.

## Styling

Hummingbird React dropdown is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available dropdown classes in the [Class overview](https://hbui.dev/docs/components/dropdown/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/dropdown/#css-variables) documentation to explore all available variables.
