## Placement

The `side` and `align` props on `DropdownMenuContent` control where the menu opens relative to its trigger.

<ComponentPreview name="dropdown-placement" className="text-center" />

## Disabled

Use `disabled` to prevent a `DropdownMenuItem` from being selected.

<ComponentPreview name="dropdown-disabled" className="text-center" />

## Non-interactive text

`DropdownMenuItemText` displays non-interactive text in a menu.

<ComponentPreview name="dropdown-item-text" className="text-center" />

## API Reference

For detailed usage guidelines, see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/dropdown-menu#api-reference) documentation.

`DropdownMenuItemText` renders non-interactive copy as a `<span>` and accepts a `className`.

## Styling

Hummingbird React dropdown is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available dropdown classes in the [Class overview](https://hbui.dev/docs/components/dropdown/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/dropdown/#css-variables) documentation to explore all available variables.
