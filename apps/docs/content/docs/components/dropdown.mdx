---
title: Dropdown Menu
description: A toggleable menu of actions or links, anchored to a trigger and positioned by Radix UI.
---

## Default

A `DropdownMenuTrigger` rendered as a `Button` opens a `DropdownMenuContent` holding a label, items, and a separator.

<ComponentPreview name="dropdown-default" />

## Groups

`DropdownMenuGroup` bundles related items under a `DropdownMenuLabel` section header.

<ComponentPreview name="dropdown-groups" />

## Placement

The `side` and `align` props on `DropdownMenuContent` control where the menu opens relative to its trigger.

<ComponentPreview name="dropdown-placement" />

## Disabled

Adding `disabled` to a `DropdownMenuItem` prevents it from being selected.

<ComponentPreview name="dropdown-disabled" />

## Non-interactive text

`DropdownMenuItemText` renders plain, non-actionable copy inside the menu.

<ComponentPreview name="dropdown-item-text" />

## API Reference

For detailed usage guidelines, see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/dropdown-menu#api-reference) documentation.

`DropdownMenuItemText` renders non-interactive copy as a `<span>` and accepts a `className`.

## Styling

Hummingbird React dropdown is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available dropdown classes in the [Class overview](https://hbui.dev/docs/components/dropdown/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/dropdown/#css-variables) documentation to explore all available variables.
