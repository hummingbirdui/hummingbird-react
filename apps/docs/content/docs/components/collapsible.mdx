---
title: Collapsible
description: An interactive component that expands and collapses a single panel of content.
---

## Default

A collapsible pairs a trigger with one content panel that it shows and hides, animating the panel's height open and closed.

<ComponentPreview name="collapsible-default" />

## Open By Default

Add `defaultOpen` to render the panel expanded on first paint.

<ComponentPreview name="collapsible-default-open" />

## Controlled

Pass `open` together with `onOpenChange` to control the open state from the parent.

<ComponentPreview name="collapsible-controlled" />

## Disabled

Set `disabled` on the root to prevent the panel from being toggled.

<ComponentPreview name="collapsible-disabled" />

## API Reference

For detailed usage guidelines, see the [Radix UI](https://www.radix-ui.com/primitives/docs/components/collapsible#api-reference) documentation.

## Styling

Collapsible is a behavior-only primitive and ships no Hummingbird classes of its own. The panel's height is animated through `data-state`; the trigger and content take any Hummingbird utility classes as needed.
