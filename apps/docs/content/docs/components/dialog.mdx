---
title: Dialog
description: A modal dialog built on Radix UI that traps focus, locks scroll, and dismisses on backdrop click or Escape.
---

## Default

A `DialogTrigger` opens the `DialogContent`, which composes a header, body, and footer with a title, description, and close buttons.

<ComponentPreview name="dialog-default" />

## Centered

Setting `centered` on `DialogContent` vertically centers the dialog in the viewport.

<ComponentPreview name="dialog-centered" />

## Sizes

The `size` prop sets the dialog width across `"sm"`, `"md"`, `"lg"`, and `"xl"`.

<ComponentPreview name="dialog-sizes" />

## Scrollable

Setting `scrollable` keeps the header and footer fixed while a long body scrolls inside the dialog.

<ComponentPreview name="dialog-scrollable" />

## Fullscreen

Setting `fullscreen` expands the dialog to fill the entire viewport.

<ComponentPreview name="dialog-fullscreen" />

## Controlled

Passing `open` and `onOpenChange` to the `Dialog` root drives its open state externally.

<ComponentPreview name="dialog-controlled" />

## Styling

Hummingbird React dialog is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available dialog classes in the [Class overview](https://hbui.dev/docs/components/dialog/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/dialog/#css-variables) documentation to explore all available variables.
