---
title: Accordion
description: A vertically stacked set of interactive headings that each reveal a section of content.
---

## Default

Only one panel is open at a time. Setting `collapsible` allows the open panel to be closed again.

<ComponentPreview name="accordion-default" />

## Multiple

Use `type="multiple"` to let any number of panels stay open at once.

<ComponentPreview name="accordion-multiple" />

## Disabled

Add `disabled` to an `AccordionItem` to prevent it from opening, or to the
`Accordion` root to disable every item.

<ComponentPreview name="accordion-disabled" />

## API Reference

Built on the [Radix UI Accordion](https://www.radix-ui.com/primitives/docs/components/accordion) primitive. Each part forwards all props to its Radix counterpart.

## Styling

Hummingbird React accordion is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available accordion classes in the [Class overview](https://hbui.dev/docs/components/accordion/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/accordion/#css-variables) documentation to explore all available variables.
