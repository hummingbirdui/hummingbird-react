title: Badge
description: A small count or status descriptor for UI elements.
---

## Default

The default badge is filled with the primary color.

<ComponentPreview name="badge-default" />

## Variants

The `variant` prop changes the visual style: `filled` (default), `subtle`, or `outline`.

<ComponentPreview name="badge-variants" />

## Colors

The `color` prop conveys intent with `primary` (default), `neutral`, `secondary`, `info`, `success`, `warning`, or `danger`.

<ComponentPreview name="badge-colors" />

## Sizes

The `size` prop offers three sizes: `sm` (default), `md`, and `lg`.

<ComponentPreview name="badge-sizes" />

## Link

The `link` prop styles the badge as an interactive link.

<ComponentPreview name="badge-link" />

## With action button

`BadgeActionButton` adds a trailing button.

<ComponentPreview name="badge-with-action" />

## Styling

Hummingbird React badge is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available badge classes in the [Class overview](https://hbui.dev/docs/components/badge/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/badge/#css-variables) documentation to explore all available variables.
