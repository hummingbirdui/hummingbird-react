---
title: List Group
description: A flexible container for displaying a series of related content as a vertical list.
---

## Default

A `ListGroup` renders a vertical list of `ListGroupItem` elements.

<ComponentPreview name="list-group-default" />

## Active and disabled

The `active` prop marks the current item and `disabled` mutes it and blocks interaction.

<ComponentPreview name="list-group-active-disabled" />

## Actionable

The `action` prop styles an item as interactive, and `asChild` renders it as a link or button.

<ComponentPreview name="list-group-action" />

## Numbered

The `numbered` prop prefixes each item with an incrementing counter.

<ComponentPreview name="list-group-numbered" />

## Pinned

The `pinned` prop keeps an item stuck to the top while the list scrolls.

<ComponentPreview name="list-group-pinned" />

## Custom content

`ListText` provides title and subtitle text for composing richer item layouts.

<ComponentPreview name="list-group-custom-content" />

## Styling

Hummingbird React list group is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available list group classes in the [Class overview](https://hbui.dev/docs/components/list-group/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/list-group/#css-variables) documentation to explore all available variables.
