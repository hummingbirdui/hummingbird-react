## Default

The final item marks the current page with `active` and a `BreadcrumbPage`.

<ComponentPreview name="breadcrumb-default" />

## Separators

The `separator` prop on `BreadcrumbList` changes the divider between items: `slash` (default), `dashed`, or `arrow`.

<ComponentPreview name="breadcrumb-separators" />

`Breadcrumb`, `BreadcrumbLink`, and `BreadcrumbPage` render a `<nav>`, `<a>`, and `<span>`. Each accepts `asChild` and `className`.

## Styling

Hummingbird React breadcrumb is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available breadcrumb classes in the [Class overview](https://hbui.dev/docs/components/breadcrumb/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/components/breadcrumb/#css-variables) documentation to explore all available variables.
