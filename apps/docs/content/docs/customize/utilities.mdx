---
title: Utilities
description: Frequently used utilities for customizing Hummingbird React components.
---

All Tailwind utility classes can be applied to Hummingbird React components through the `className` prop. Custom classes are merged with the component classes, so utilities can adjust spacing, typography, radius, and more without writing custom CSS. This page lists the most frequently used utilities; see the <a href="https://tailwindcss.com/docs" target="_blank" rel="noopener noreferrer">official Tailwind CSS docs</a> for the complete reference.

Each utility supports these value forms:

- `p-4` — a number, multiplied by the `--spacing` scale (where applicable)
- `p-px` — a fixed `1px` value
- `p-(--my-padding)` — a custom CSS property
- `p-[12px]` — an arbitrary value
