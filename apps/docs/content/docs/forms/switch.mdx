## Default

A `label` renders a complete, clickable field.

<ComponentPreview name="switch-default" />

## Colors

The `color` prop offers `primary` (default), `secondary`, `success`, `info`, `warning`, `danger`, and `neutral`.

<ComponentPreview name="switch-colors" />

## Sizes

The `size` prop offers three sizes: `sm`, `md` (default), and `lg`.

<ComponentPreview name="switch-sizes" />

## Inline

The `inline` prop lays fields out in a row.

## Styling

Hummingbird React switch is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available switch classes in the [Class overview](https://hbui.dev/docs/forms/switch/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/switch/#css-variables) documentation to explore all available variables.
