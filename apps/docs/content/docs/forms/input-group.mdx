## Default

Use `InputGroupText` to display text or icons inside an `InputGroup`.

## Sizes

The `size` prop offers three sizes: `sm`, `md` (default), and `lg`.

## Styling

Hummingbird React input group is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available input group classes in the [Class overview](https://hbui.dev/docs/forms/input-group/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/input-group/#css-variables) documentation to explore all available variables.
