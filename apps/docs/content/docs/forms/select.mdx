## Default

`Select` renders a native `<select>` and takes `<option>`s as children.

<ComponentPreview name="select-default" className="max-w-sm mx-auto" />

## Variants

The `variant` prop switches between `outline` (default, bordered) and `fill` (a filled surface).

<ComponentPreview name="select-variants" />

## Sizes

The `size` prop offers three sizes: `sm`, `md` (default), and `lg`.

<ComponentPreview name="select-sizes" />

## Colors

The `color` prop sets the focus accent: `secondary`, `info`, `success`, or `warning`.

<ComponentPreview name="select-colors" />

## Validation state

The `state` prop marks the select `valid` or `invalid`, and pairs with a `FormText` of the matching `variant`. The `invalid` state also sets `aria-invalid`.

<ComponentPreview name="select-validation" />

## Styling

Hummingbird React select is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available select classes in the [Class overview](https://hbui.dev/docs/forms/select/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/select/#css-variables) documentation to explore all available variables.
