## Styling

Hummingbird React form range is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available form range classes in the [Class overview](https://hbui.dev/docs/forms/form-range/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/form-range/#css-variables) documentation to explore all available variables.
