## Colors

The `color` prop offers `primary` (default), `secondary`, `success`, `info`, `warning`, `danger`, and `neutral`.

<ComponentPreview name="checkbox-colors" />

## Sizes

The `size` prop offers three sizes: `sm` (default), `md`, and `lg`.

<ComponentPreview name="checkbox-sizes" />

## Inline

The `inline` prop lays fields out in a row.

<ComponentPreview name="checkbox-inline" />

## Indeterminate

The `indeterminate` prop renders the mixed state, such as a "select all" parent.

## Styling

Hummingbird React checkbox is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available checkbox classes in the [Class overview](https://hbui.dev/docs/forms/checkbox/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/checkbox/#css-variables) documentation to explore all available variables.
