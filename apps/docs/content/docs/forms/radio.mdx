## Default

`RadioGroup` groups related radio buttons and manages the selected value.

<ComponentPreview name="radio-default" />

## Colors

The `color` prop offers `primary` (default), `secondary`, `success`, `info`, `warning`, `danger`, and `neutral`.

<ComponentPreview name="radio-colors" />

## Sizes

The `size` prop offers three sizes: `sm` (default), `md`, and `lg`.

<ComponentPreview name="radio-sizes" />

## Inline

The `inline` prop lays fields out in a row.

## Styling

Hummingbird React radio is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available radio classes in the [Class overview](https://hbui.dev/docs/forms/radio/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/radio/#css-variables) documentation to explore all available variables.
