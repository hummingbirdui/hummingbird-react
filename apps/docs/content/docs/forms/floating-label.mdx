## Default

`FloatingLabel` wraps a single control and floats its `label` once the control is focused or filled.

## Styling

Hummingbird React floating label is styled entirely through Hummingbird's utility classes and CSS
variables.

- See the full list of available floating label classes in the [Class overview](https://hbui.dev/docs/forms/floating-label/#class-overview).
- Visit the [CSS Variables](https://hbui.dev/docs/forms/floating-label/#css-variables) documentation to explore all available variables.
